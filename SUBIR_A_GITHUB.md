# Subir este proyecto a GitHub

Este proyecto ya tiene Git inicializado y un primer commit creado.

1. Crea un repositorio vacío en GitHub llamado `jldv1508-4089`.
2. No marques README, `.gitignore` ni licencia al crearlo.
3. Copia la URL HTTPS del repositorio.
4. Desde esta carpeta, ejecuta:

```bash
git remote add origin https://github.com/TU-USUARIO/jldv1508-4089.git
git push -u origin main
```

Después, en Vercel:

1. Importa el repositorio `jldv1508-4089`.
2. Framework: Next.js.
3. Añade la variable `ADMIN_PASSWORD`.
4. Despliega.
