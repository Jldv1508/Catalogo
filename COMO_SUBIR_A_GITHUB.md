# Como subir a GitHub

Sube esta carpeta descomprimida: `github-jldv1508`.

No subas los ZIP grandes de `proyecto-jldv1508-final`; GitHub web rechaza archivos grandes.

Esta carpeta ya esta preparada:

- Sin `node_modules`
- Sin `.next`
- Sin archivos `.zip`
- Sin archivos individuales mayores de 25 MB
- Con 972 imagenes unicas en `public/imagenes`

En Vercel configura:

```text
ADMIN_USER=admin
ADMIN_PASSWORD=tu_contrasena_segura
```

La web publica estara en `/` y el renombrador privado en `/admin`.
