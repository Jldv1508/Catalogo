# Subir imagenes a GitHub por lotes

La carpeta de imagenes esta dividida para cumplir los limites de GitHub web:

- Menos de 100 archivos por lote.
- Menos de 25 MB por lote.

Ruta:

```text
public/imagenes/lote-001
public/imagenes/lote-002
...
public/imagenes/lote-021
```

Si usas la web de GitHub, sube primero los archivos del proyecto sin `public/imagenes`.
Luego entra en `public/imagenes` y sube cada carpeta `lote-XXX` por separado.

La forma mas facil sigue siendo GitHub Desktop, porque permite subir todo el repositorio sin estos cortes manuales.
